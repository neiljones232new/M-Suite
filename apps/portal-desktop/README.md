# M-Suite Portal Desktop (Tauri)

Native macOS desktop app for managing M-Suite services.

## Features

- Start, stop, and restart each external service:
  - Practice Web (`3000`)
  - Practice API (`3001`)
  - Customs UI (`5173`)
  - Customs Backend (`3100`)
- Start, stop, and restart the full suite.
- Live port monitoring with PID visibility.
- Background process logs at `/tmp/msuite-*.log`.

## Prerequisites

- Rust toolchain installed (`rustup`, `cargo`, `rustc`)
- Xcode Command Line Tools
- Node + pnpm

## Run in development

```bash
pnpm install
pnpm portal:desktop
```

## Build macOS app bundle

```bash
pnpm portal:desktop:build
```

Output bundle is generated by Tauri under `src-tauri/target/release/bundle/macos`.
